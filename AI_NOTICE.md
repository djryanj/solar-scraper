# AI Notice

This repository may contain code, documentation, tests, or workflow changes created or edited with the assistance of AI tools.

## Expectations

- AI-assisted changes must still be reviewed by a human maintainer.
- AI output is treated as proposed code, not trusted code.
- Security-sensitive, release, infrastructure, and dependency changes require explicit human review.

## Scope

AI assistance may be used for:

- drafting documentation
- generating tests
- refactoring code
- proposing CI or automation updates

AI assistance should not replace repository ownership, review, or release approval.
